#### 문제 설명
>주어진 $1 \times d$ 배열을 $n \times \frac{d}{n}$ 차원 행렬로 만들어라

#### Code
```python
def solution(num_list, n):
	# loop range와 start, end 모두 신경써야 한다.
    answer = []
    for i in range(0,len(num_list)//n):
        start = i*n
        end = (i+1)*n
        answer.append(num_list[start:end])
        
    return answer
```
