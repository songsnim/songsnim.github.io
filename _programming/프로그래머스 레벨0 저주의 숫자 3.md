#### 문제 설명
3x 마을 사람들은 3을 저주의 숫자라고 생각하기 때문에 숫자에 3이 보이거나, 3의 배수면 사용하지 않는다.
```
````

#### Code
```python
def solution(n):
	# 저주의 숫자면 count를 하지 않고, 아닐 때만 count를 해서 count가 n에 도착하는 순간
	# 그 순간까지 갈 때 loop 돈 횟수가 정답이다.
    cnt = 0
    answer = 0
    while cnt != n:
        answer += 1 
        if answer % 3 != 0 and '3' not in str(answer):
            cnt += 1
            
    return answer
```
